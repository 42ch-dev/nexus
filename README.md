# Nexus

Nexus is a local-first creative writing tool.

## Quick Start

```bash
# Build
cargo build --release

# Show help
./target/release/nexus42 --help

# Start daemon
./target/release/nexus42 daemon start
```

## Components

| Component | Description |
|-----------|-------------|
| `nexus42` | CLI client with integrated daemon runtime |
