# Metadata Extraction

For each reference source that has been text-extracted, identify and catalog the following metadata:

- **Title**: document title or filename
- **Author**: author name(s) if available
- **Date**: publication or creation date if available
- **Type**: source type (pdf, markdown, text, url, html)
- **Summary**: one-paragraph summary of the content
- **Key topics**: 3-5 main topics or themes

Format as structured entries, one per source.
