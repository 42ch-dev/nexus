# Synthesizing Exit Check

Evaluate whether the synthesizing phase has produced a complete research report.

Review the synthesized output. If the report contains all five required sections
(Executive Summary, Source Inventory, Thematic Analysis, Key Findings, Gaps & Recommendations),
respond with "go" to proceed.

If the report is incomplete, missing sections, or the content is too shallow to be useful,
respond with "wait" and specify which sections need improvement.
