## Objective

Create `crates/nexus-daemon-runtime` as a library crate by migrating daemon internals from `crates/nexus42d/src/*`. Preserve all existing behavior — this batch does NOT change any user-facing behavior.

## Scope

### In Scope

1. Create new crate `crates/nexus-daemon-runtime/` (library only, no binary)
2. Migrate modules from `nexus42d`:
   - `api/` → runtime API handlers
   - `auth/` → runtime auth
   - `db/` → runtime DB integration
   - `lifecycle/` → runtime lifecycle state machine
   - `workspace/` → runtime workspace management
   - `cli_config.rs` → `config.rs` (renamed, daemon-specific naming removed)
3. Copy dependency set from `nexus42d/Cargo.toml`
4. Add `crates/nexus-daemon-runtime` to workspace members in root `Cargo.toml`
5. Add `AGENTS.md` for the new crate
6. Keep `nexus42d` compiling as-is (temporary parallel compile path)

### Out of Scope

- CLI wiring changes (Batch 2)
- Removing `nexus42d` (Batch 3)
- Naming/log changes (Batch 4)
