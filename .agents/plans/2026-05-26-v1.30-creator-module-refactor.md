`crates/nexus42/src/commands/creator/mod.rs` is ~3200 lines and includes KB queue/status/list handlers inline. QC R19: extract KB commands to a dedicated submodule for maintainability before further Author Intelligence growth.

**No behavior change** — refactor only unless a bug is found during move.

---

## 2. Goals

### C1 — Module extraction (R19)

1. Add `crates/nexus42/src/commands/creator/kb.rs` (or `kb/mod.rs` if multiple files needed).

2. Manual smoke: `nexus42 creator kb --help`, `queue-extract --help`, `extract-status --help`.
