## V2 target

6 top-level command groups:

1. `daemon` — lifecycle + orchestration
2. `acp` — ACP capability plane
3. `creator` — identity + knowledge assets
4. `sync` — platform sync
5. `platform` — platform operations
6. `system` — diagnostics + config
