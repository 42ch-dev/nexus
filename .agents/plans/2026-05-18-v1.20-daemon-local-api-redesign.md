## Objective

Redesign the daemon local HTTP API to be a clean, reusable local control plane. Five key changes:

1. Replace Platform OAuth auth with `NEXUS42_DAEMON_API_KEY` (env-based, localhost bypass when empty)
2. Remove world/explore platform-proxy routes
3. Remove ACP-specific routes, absorb into agent-host abstraction
4. Migrate CLI-direct operations (workspace, creator, preset, KB) to daemon API
5. Polish API for web UI readiness (error format, pagination)
