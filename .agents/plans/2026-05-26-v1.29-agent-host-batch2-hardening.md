## 2. Goals

### Step 0 — Audit (mandatory first PR slice)

1. Document which `TimeoutConfig` fields are enforced per provider (`acp.rs`, `claude.rs`, `manager.rs`).
2. Update tracker DF-21 row to reflect **actual gaps** (e.g. missing `session_ms` / `initialize_ms` bounds) before coding.

### Batch 2 implementation

1. **DF-21**: Enforce remaining timeout stages on all provider exec paths; tests with short timeouts.
2. **DF-22**: Implement heuristic or rules-based `ToolRiskClassifier` beyond static deny list (document rules in agent-host spec).
3. **DF-23**: Wire ACP streaming events to host `StreamingChunk` relay (minimal happy path + test).
4. **DF-27**: Validate session UUID on `/v1/local/agent-host/sessions/{id}/*` — 400 on malformed.
5. **DF-28**: Reject `..` and non-canonical paths in config resolution.
6. **R11/R12**: Implement `Drop` kill + cancel drain for persistent native child where feasible; else defer with residual note.
