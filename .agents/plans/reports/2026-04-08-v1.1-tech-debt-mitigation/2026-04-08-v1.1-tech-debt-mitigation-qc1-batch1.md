### 🟡 Warning

**W1: PolicyAwareClientHandler interactive prompting not implemented (ACP-R7)**
- **Issue**: `prompt_user()` method has TODO comment and defaults to `PermissionDecision::Deny` in automated environments (line 214-224, client.rs). This is documented behavior for V1.1 but limits usability in interactive sessions.
- **Impact**: Users cannot grant permissions interactively; defaults to deny for safety.
- **Fix**: Add proper interactive prompting library (dialoguer) for V1.2+.
- **Source**: Manual code review, `crates/nexus42/src/acp/client.rs:214-224`
- **Confidence**: High
- **Cross-Reference**: ACP-R7 implementation note in plan

**W2: SessionManager::default_sessions_file() uses expect() in non-test code**
- **Issue**: Line 56 uses `dirs::home_dir().expect("Failed to get home directory")`. While this is initialization context, it could panic in unusual environments (containerized, restricted home access).
- **Impact**: Low - this is called during setup, not in hot path.
- **Fix**: Consider returning Result in constructor or documenting the panic precondition.
- **Source**: `crates/nexus42/src/acp/session_manager.rs:56`
- **Confidence**: High

**W3: Missing validation for agent_id and workspace_hint in SessionEntry**
- **Issue**: `SessionEntry` accepts any string for `agent_id` and any `PathBuf` for `workspace_hint` without validation. Malicious or malformed values could be persisted.
- **Impact**: Medium - persisted data used for session matching, but scope is local user workspace.
- **Fix**: Add validation in `SessionEntry::new()` constructor (not currently implemented) or document trust boundary.
- **Source**: `crates/nexus42/src/acp/session_manager.rs:18-30`
- **Confidence**: Medium

**W4: Daemon tool execution lacks rate limiting**
- **Issue**: `/v1/local/acp/tool/execute` endpoint has no rate limiting. Agent could flood the daemon with file operations.
- **Impact**: Low - daemon is local-only, same-user context.
- **Fix**: Add configurable rate limiting for V1.2+ if needed.
- **Source**: `crates/nexus42d/src/api/handlers/acp.rs:63-88`
- **Confidence**: Medium
