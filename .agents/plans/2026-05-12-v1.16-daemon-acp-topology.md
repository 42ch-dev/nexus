# Plan: V1.16 Daemon and ACP Top-Level Command Topology

**Plan ID**: `2026-05-12-v1.16-daemon-acp-topology`
**Status**: InProgress → Done (pending QC)
**Working branch**: `feature/v1.16-daemon-acp-topology`
**Theme**: B1 — Command surface refactor
**Milestone**: M2

## Purpose

Restructure the CLI command surface so that:
1. `nexus42 daemon` gains V2 subcommands: `restart`, `logs`, `doctor`, and `orchestrate` sub-group
2. A new `nexus42 acp` top-level command group is created
3. Old `nexus42 agent`, `session`, `policy`, `permission` are hidden from top-level

- [x] T1: Create `commands/acp.rs` — new ACP command module with all subcommands
