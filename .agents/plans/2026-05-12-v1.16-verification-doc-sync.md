## Done definition to verify

From `.agents/knowledge/v1.16-delivery-compass-v1.md` §5, V1.16 is Done only if all six items are verified:

1. `creator use <creator_id_or_handle>` persists a normalized `creator_id`.
2. `creator status/list` stably output `creator_id`, `handle`, and `display_name`.
3. `daemon orchestrate` and `acp` have no responsibility overlap.
4. ACP execution path coverage is 100%, with no bypass path.
5. `creator kb` default `work` is usable and `world` mapping is usable.
6. User-path / Creator-path auth behavior matches `auth-session-model-v1`.
