## 1. Problem Statement

V1.27 made four-domain `assemble-moment` persistent, but:

1. World KB assembly uses unbounded [`list_by_world`](../../crates/nexus-moment-context-assembly/src/moment.rs) ŌĆö no `KbQuery` filters.
2. There is no cross-domain token budget after Stage0 truncation.
3. Authors must choose between **`assemble-local`** (Stage0 / TwoStage / degradation) and **`assemble-moment`** (SQLite domains) ŌĆö split mental model and spec drift.

Pre-release policy allows breaking CLI consolidation (L3).

---

## 2. Goals

### Theme L3 ŌĆö Single CLI assembly command

1. Remove `ContextCommand::AssembleLocal` and the public `assemble-local` subcommand.
2. Extend `AssembleMoment` with flags (frozen names):
   - `--max-tokens <usize>` ŌĆö cross-domain budget
   - `--no-fragments` ŌĆö inverse of current `include_fragments` default true
   - `--hint <text>` ŌĆö Stage0 / two-stage hint
   - `--kb-limit`, `--kb-search`, `--kb-type` ŌĆö World KB query (require `--world-id` when KB filters used)
   - `--knowledge-limit` ŌĆö User knowledge list cap (default 20)
3. Preserve runtime behavior from `assemble_local_with_routing`:
   - `local_only` ŌåÆ Stage0 + moment domains
   - `local_first` / `cloud_enhanced` ŌåÆ degradation guard + TwoStage attempt with Stage0 fallback (platform cloud assemble remains deferred)
4. Update deferred `assemble` error text to mention **`assemble-moment` only**.
5. Update [`command_surface_contract.rs`](../../crates/nexus42/tests/command_surface_contract.rs) and `context.rs` tests.

### Theme C2 ŌĆö Query and budget in `nexus-moment-context-assembly`

1. Replace `fetch_world_kb` `list_by_world` with `KbStore::query` + `KbQuery` (limit, text_search, block_type).
2. Plumb query parameters from `MomentRequest` (new optional fields) or from CLI-built request.
3. Implement cross-domain truncation in `MomentContext::to_full_context` or dedicated helper:
   - Never truncate personality section inside Stage0 (existing rule)
   - Apply remaining budget to world_kb, user_knowledge, timeline after stage0
4. Document section priority in crate module doc or AGENTS.md (this plan touches spec paragraph only; full cli-spec in SSOT plan).

### Theme C2 ŌĆö Tests

1. Unit/integration: seeded DB + `--kb-limit 1` yields single KB line in output.
2. Persistent E2E: `demo seed` ŌåÆ `assemble-moment` four domains (extend V1.27 C3.1).
3. No `rg AssembleLocal` in `crates/nexus42/src/commands/platform/context.rs` production paths.
