## 0) Scope lock

This compass covers **removal of all deprecated CLI top-level commands, deprecated command implementations, deprecated runtime warnings, obsolete test helpers, and other dead code accumulated across V1.2‚ÄìV1.18 refactoring**.

**Product is not yet released** ‚Äî all changes are breaking and require no backward compatibility.

### In scope

1. Remove all deprecated top-level commands from `clap` app in `cli.rs`
2. Delete deprecated command implementation modules (`commands/init.rs`, `commands/auth.rs`, `commands/db.rs`, `commands/debug.rs`, etc.)
3. Remove deprecated sub-commands from existing command groups (`creator` soul/memory, `acp` agent-use)
4. Remove `#[deprecated]` functions (`ContextSummaryBuilder::with_default_config` in `context/summary.rs`)
5. Remove deprecated runtime warning `eprintln!` redirects in `main.rs`
6. Delete test functions that only test deprecated behavior
7. Clean up dead imports after deletions

### Out of scope

- New feature work (capabilities, orchestration stubs ‚Äî tracked as DF-29‚ÄìDF-41)
- V1.19 hardening backlog (DF-18‚ÄìDF-28)
- CLI command implementation improvements (only removal)
- Any schema/contract changes
