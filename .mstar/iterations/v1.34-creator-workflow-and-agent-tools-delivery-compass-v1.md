## 0. Position

V1.33 shipped the **narrative Work loop** (`creator run`, Creative Brief Intake, `novel-writing`, memory review, `llm_judge` fix). V1.34 extends that foundation into a **feature-complete generic creator workflow** (FL-E) and opens a **mediated Agent tool surface** so external ACP agents can call selected Nexus capabilities without spawning `nexus42` or bypassing daemon policy.

The locked axis is:

> **FL-E stage-complete Work journey + explicit stage gates + minimal `nexus.*` host tools parallel to preset orchestration.**

This is **not** a conditional routing / branching engine iteration (see FL-D / DF-56). It is **not** full `acp-capability-set` parity (see DF-46).

When this compass conflicts with an active locked plan, the active plan wins for delivery batching.

## 2. Architecture

### 2.1 FL-E on Work (stage model)

```text
creator run start  →  creative-brief-intake (work_init)  →  intake_status=complete
        │
        ▼
creator run stage advance --stage research   →  research (knowledge_ingest + work_continue)
        ▼
creator run stage advance --stage produce    →  novel-writing (work_continue)
        ▼
creator run stage advance --stage review     →  reflection-loop (work_continue)
        ▼
creator run stage advance --stage persist    →  memory review + kb-extract / queue-extract
```

Normative detail: [creator-workflow.md](../knowledge/specs/creator-workflow.md).

### 2.2 Dual channel: orchestration vs Agent tools

| Channel | Initiator | Path | Use |
| --- | --- | --- | --- |
| **Orchestration** | Schedule / preset graph | `acp.prompt`, `judge.*`, `creator.*` capabilities → worker IPC | Deterministic multi-step presets |
| **Agent tools** | External ACP Agent tool call | `worker/agent_tool_request` or HTTP → `HostToolExecutor` → Local API handlers | LLM-driven context gather, Work read/update |

Agents **must not** spawn `nexus42` for tool execution. Shared handler kernel is normative in [agent-nexus-tool-bridge.md](../knowledge/specs/agent-nexus-tool-bridge.md).

### 2.3 V1.34 minimal tool set

| Tool ID | RW | Notes |
| --- | --- | --- |
| `nexus.context.whoami` | R | Active creator |
| `nexus.workspace.info` | R | Workspace root, flags |
| `nexus.work.get` | R | Work by `work_id` |
| `nexus.work.patch` | W | Append inspiration / stage metadata only |
| `nexus.orchestration.schedule_status` | R | Linked schedules for Work |
| `nexus.context.assemble` | R | Local/read-only or `policy_blocked` |
