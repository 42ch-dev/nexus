## 2. Architecture

### 2.1 Author intelligence (FL-A)

```text
ACP session / CLI
        │
        ▼
session_capture ──POST──► /v1/local/memory/pending-review
        │
        ▼
creator memory pending list | review
        │
        ▼
long-term memory + SOUL (files + optional soul meta in state.db)

Schedule / preset ──► soul.experience.aggregate (capability)
        │
        ▼
SOUL.md ## Experience

assemble-moment ──► Stage0 (delimiter-based personality split) + four domains
```

### 2.2 KB extract (FL-B)

```text
creator kb queue-extract ──► kb_extract_jobs (state.db)
        │
        ▼
daemon schedule / orchestrate run ──► preset kb-extract
        │
        ├── capability kb.extract_work
        ├── acp_prompt (real IPC)
        └── SqliteKbStore insert (world scope)

creator kb extract-status ◄── job row
```

**Invariant**: No `nexus42` CLI subcommand invokes an LLM provider directly for extraction.
