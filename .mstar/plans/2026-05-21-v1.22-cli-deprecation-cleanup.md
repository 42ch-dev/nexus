## Scope

Remove all deprecated CLI commands, their implementations, deprecated runtime redirects, obsolete tests, and dead imports. Product is pre-release — no backward compatibility required.

**Complexity**: Medium — some deprecated modules contain shared utilities actively used by replacement commands. These must be extracted before deletion.
