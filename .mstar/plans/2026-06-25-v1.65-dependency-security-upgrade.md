## Overview

Close the 5 open dependabot advisories (9 alerts across manifests) on `main`. All are **dev/test-scope only** — zero runtime/production exposure (the shipped `nexus42` binary embeds a static rust-embed dist; vite/vitest/wiremock are devDependencies). The fixes are major-version bumps (vitest 2→3, vite 5→6) + a transitive Rust dev-dep cleanup. Because vitest 2→3 is a major bump, this plan **must merge before P1** so the test baseline is written on vitest 3.x (avoids rework).
