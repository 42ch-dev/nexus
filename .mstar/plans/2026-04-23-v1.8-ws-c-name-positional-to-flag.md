## §1 Overview

Change `creator register <name>` to `creator register --name <name>` for consistency with the frozen spec. This is a breaking change for existing user scripts using positional argument syntax.

**Compass**: [v1.8-delivery-compass-v1.md](.agents/iterations/v1.8-delivery-compass-v1.md)

**Effort (agent-oriented)**: XS (~0.5 agent session)
**Dependencies**: None — can begin immediately.

---

## §2 Scope

### In scope

- Modify clap definition in `crates/nexus42/src/commands/creator.rs` from positional `name` to optional `--name` flag.
- Update help text to match frozen spec: `nexus42 creator register --name "My Agent" [--source cli|web_agent] [--handle <handle>]`.
- Update E2E tests in `crates/nexus42/tests/creator_register_e2e.rs` to use new syntax.
- Ensure positional syntax produces a clear error message with migration hint.

### Out of scope

- Other subcommand argument changes.
