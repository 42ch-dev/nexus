# V1.4 WS5 — `schemas/` boundary refactor

> **Historical plan (Done).** OSS `schemas/` layout after V1.21: [schemas-directory-layout.md](../knowledge/specs/schemas-directory-layout.md) (`cloud-sync/`, no `meta/`). Audit table paths in [archived/knowledge/schemas-boundary.md](../archived/knowledge/schemas-boundary.md) use `cloud-sync/` where updated 2026-05-20.

> **For agentic workers:** Ordered checklist. Commit after each task unless noted.

**Plan ID**: `2026-04-17-v1.4-ws5-schemas-boundary`
**Wave**: V1.4 WS5 — see [.agents/iterations/v1.4-delivery-compass-v1.md](../iterations/v1.4-delivery-compass-v1.md) §4 WS5
**Wave-0 spec (authoritative)**: [archived/knowledge/schemas-boundary.md](../archived/knowledge/schemas-boundary.md) (§5 audit procedure + §7 migration steps)
**Working branch**: `create feature/v1.4-ws5-schemas-boundary from main` — **fully parallel with all other WS**; `schemas/acp-runtime/` since removed (types in `src/local/acp_runtime/`).

**Goal**: Audit every `schemas/**/*.schema.json` against the wire rule ([archived/knowledge/schemas-boundary.md](../archived/knowledge/schemas-boundary.md) §2.1), move clearly-local schemas into hand-written Rust under `crates/nexus-contracts/src/local/`, regenerate contracts, and bump the npm package minor.

**Architecture**:
- **Wire criterion**: schema stays in `schemas/` iff `nexus-platform` observes it (platform HTTP bodies, sync bundle/delta/outbox payloads). Else: hand-written Rust under `crates/nexus-contracts/src/local/`, **not** codegen'd.
- **Placement** per §3: `src/local/<module>/*.rs`, one Rust file per schema, `serde::{Serialize, Deserialize}` derives, unit tests colocated.
- **Size guard** per §8 / compass RISK-06: if audit shows >30% of `schemas/domain/*` must stay wire, WS5 is rescoped mid-flight to ship only the clearly-local subset; rest deferred to V1.5.

**Tech Stack**: Rust hand-written types, `serde`, `serde_json` for round-trip tests, no new external dependencies.

## Task 3 — Move local schemas to Rust types

**Files**:
- New: `crates/nexus-contracts/src/local/mod.rs` + submodules.
- Modify: `crates/nexus-contracts/src/lib.rs` (`pub mod local;`).
- Delete: each classified-local `.schema.json` file.

- [ ] **Step 1: Create the `local/` subtree**

```bash
mkdir -p crates/nexus-contracts/src/local/{domain,acp_runtime,common}
# Or whichever sub-modules your audit requires.
```

`src/local/mod.rs`:

```rust
//! Hand-written local types per `.agents/archived/archived/knowledge/schemas-boundary.md` §3.
//! These types are NOT codegen'd; edit by hand.

pub mod domain;
pub mod acp_runtime;
// Add more as audit dictates.
```

Add `pub mod local;` to `src/lib.rs`.

- [ ] **Step 2: For each local row in the audit table, transcribe schema → Rust**

Template per file:

```rust
// crates/nexus-contracts/src/local/<area>/<name>.rs
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq)]
pub struct <TypeName> {
    pub <field>: <ty>,
    // ...
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn roundtrip_json_representative_instance() {
        let v = <TypeName> { /* fill in */ };
        let s = serde_json::to_string(&v).unwrap();
        let back: <TypeName> = serde_json::from_str(&s).unwrap();
        assert_eq!(back, v);
    }
}
```

Follow existing `schemas/<area>/<name>.schema.json` exactly (field names, types, nullable → `Option<T>`, arrays → `Vec<T>`, enums → Rust `enum` with `#[serde(rename_all = "snake_case")]` or per schema).

- [ ] **Step 3: Migrate Rust call sites from generated to local**

For each moved type, find call sites:

```bash
rg -n 'use nexus_contracts::<generated-type>' crates/ | tee /tmp/ws5-callsites.txt
```

Replace `use nexus_contracts::<GeneratedType>` with `use nexus_contracts::local::<module>::<Type>` in each call site. Confirm each crate still compiles after you finish its batch.

- [ ] **Step 4: Delete the `.schema.json` files**

For every row classified `local` in the audit:

```bash
git rm schemas/<area>/<name>.schema.json
```

- [ ] **Step 5: Compile + test**

```bash
cargo build --workspace
cargo test --workspace
cargo clippy --all -- -D warnings
cargo +nightly fmt --all -- --check
```

All green.

- [ ] **Step 6: Commit**

```bash
git add -A
git commit -m "refactor(nexus-contracts): move local-only schemas to src/local/ (WS5 T3)

- Deleted N schema files per audit.
- New hand-written Rust types under src/local/{domain,acp_runtime,...}/.
- Migrated M call sites from nexus_contracts::<Type> to nexus_contracts::local::<module>::<Type>.
- Per archived/archived/knowledge/schemas-boundary.md §2–§3."
```
