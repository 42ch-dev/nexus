# P1: Track A — Web App Scaffold

Bootstrap the OSS local Web UI app `apps/web`. Pure frontend (no daemon changes — those are P3). Establishes the project skeleton, design-token consumption, the Tauri-ready transport adapter, dev tooling, and CI. No screens yet (P2).

## Tasks

- [ ] T1: pnpm workspace — add `apps/*` to `pnpm-workspace.yaml` + root `package.json` `workspaces`. Create `apps/web/` with Vite + React 18 + TypeScript (`pnpm create vite apps/web --template react-ts`).
- [ ] T2: Stack install — TailwindCSS + shadcn/ui (component primitives) + TanStack Query (server state, default per compass §5 item #5) + React Router. Configure `tsconfig.json` paths.
- [ ] T3: `@42ch/nexus-contracts` workspace dep — add `"@42ch/nexus-contracts": "workspace:*"` to `apps/web/package.json`; verify codegen TS types import (Local API DTOs).
- [ ] T4: **DESIGN.md consumption** — wire Tailwind config + CSS variables from `apps/web/DESIGN.md` tokens (Color light+dark, Typography, Spacing, Elevation, Motion, Shapes). Implement dark-mode toggle (token-driven). DESIGN.md authored by @architect in Phase 2b; P1 consumes it.
- [ ] T5: **`tauri-api` adapter boundary** (compass §5 item #7) — define `NexusClient` interface (typed methods per Local API resource). Two impls: `BrowserClient` (`fetch http://localhost:<port>/v1/local/*` with cursor-pagination + ErrorResponse parsing) and stubbed `TauriClient` (`invoke(...)` — throws "not implemented in browser build" until V1.65). Core app logic depends only on the interface.
- [ ] T6: Dev config — Vite dev server with proxy: `/v1/local/*` → `http://localhost:<daemon-port>`. Document the dev workflow (start `nexus42 daemon start`, then `pnpm --filter web dev`).
- [ ] T7: App shell — root layout (sidebar nav: Works / Sessions / Schedule / Capabilities / Findings / Presets; main content area; header with daemon-health indicator). Placeholder routes for each Control Room + Setup screen (P2 fills them).
- [ ] T8: CI build job — `.github/workflows/ci.yml` add `web-build` leg: pnpm install (with cache) + `pnpm --filter web build` + `pnpm --filter web typecheck`. @ops-engineer review.
- [ ] T9: Repo hygiene — `apps/web/.gitignore` (node_modules, dist), `apps/web/README.md` (dev workflow + Tauri roadmap pointer), `apps/web/AGENTS.md` (frontend conventions, DESIGN.md authority).
- [ ] T10: Verify — `pnpm --filter web build` succeeds; `pnpm --filter web typecheck` clean; typecheck against `@42ch/nexus-contracts` types passes.

## Done when

- `apps/web` builds (`pnpm --filter web build`) + typechecks clean.
- DESIGN.md tokens consumed (light + dark mode toggles work).
- `NexusClient` interface + `BrowserClient` impl compile; `TauriClient` stub present.
- Dev proxy serves `/v1/local/*` to a running daemon; app shell renders placeholder routes.
- CI `web-build` leg green.
- `apps/web/AGENTS.md` + README present; workspace `apps/*` registered.
