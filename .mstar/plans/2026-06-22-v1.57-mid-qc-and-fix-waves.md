## 1. Scope

Meta-plan capturing the 3-wave merge + tri-review + mid-QA + fix-wave rhythm across all 4 implement plans (P0–P3). No product code.

### In scope

- Checklist for each wave: merge to integration, dispatch tri-review, collect QC reports, dispatch mid-QA, run fix-wave, re-merge.
- Evidence tracking: which QC reports exist, which fix-waves were targeted vs full tri-review, which residuals were opened.
- Gate enforcement: Wave N+1 cannot start until Wave N plans have fix-waves closed (or PM override).

### Out of scope

- Product code, spec amendments, or test code.
- QC report authoring (QC reviewers own those).
- Mid-QA authoring (@qa-engineer owns those).
