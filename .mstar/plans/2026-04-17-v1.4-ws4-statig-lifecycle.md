HTTP endpoint v2 schema lives in `schemas/acp-runtime/daemon-status-v2.schema.json` (WS5 will eventually classify this as local per schemas-boundary-v1; in WS4 we ship it under `schemas/` for initial delivery, and WS5 moves it).

**Tech Stack**: Rust, `statig = "0.3"` (with `async` feature), existing `nexus42d` axum stack, `tracing` for introspection hooks.

## Task 4 — Entry/exit actions (§5) with mockable subsystems

**Files**: `lifecycle/actions.rs`, integration points in `main.rs`.

- [ ] **Step 1: Extract a `Subsystem` trait for each managed dep**

```rust
#[async_trait]
pub trait SubsystemBootstrap: Send + Sync {
    async fn start(&self) -> anyhow::Result<()>;
    async fn shutdown(&self, grace_ms: u64) -> anyhow::Result<()>;
    async fn health(&self) -> SubsystemHealth;
}
```

Implement five of these in `crates/nexus42d/src/lifecycle/subsystems/`:
- `HttpSubsystem` (wraps the existing listener bind)
- `DbSubsystem` (wraps SQLite pool + migrations)
- `SyncSubsystem` (wraps `nexus-sync` outbox reader)
- `EngineSubsystem` (wraps `nexus-orchestration::GraphFlowEngine`)
- `WorkerMgrSubsystem` (wraps `nexus-orchestration::WorkerManager`)

- [ ] **Step 2: Write entry/exit actions per §5**

`Starting.entry` spawns five tokio tasks, each calling `SubsystemBootstrap::start` on one of the above, then dispatching `SubsystemUp(kind)` or `SubsystemFailed(...)`.

`Running.entry` starts the system preset session (if not already running).

`Stopping.entry` sends shutdown signals to every subsystem in reverse dep order; starts a watchdog for `ShutdownTimeout`.

`Failed.entry` logs final `tracing` event, waits 100 ms for log flush, calls `std::process::exit(exit_code)`.

Make each action small (<50 LOC) and delegate to the subsystem trait.

- [ ] **Step 2: Wire signals + panic**

```rust
#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let lifecycle = StatigLifecycle::new_with_real_subsystems(&config).await?;

    let lc_for_signals = lifecycle.clone();
    tokio::spawn(async move {
        let mut sigterm = tokio::signal::unix::signal(tokio::signal::unix::SignalKind::terminate()).unwrap();
        let mut sigint  = tokio::signal::unix::signal(tokio::signal::unix::SignalKind::interrupt()).unwrap();
        tokio::select! {
            _ = sigterm.recv() => lc_for_signals.dispatch(Event::ShutdownRequested { source: "signal".into() }),
            _ = sigint.recv()  => lc_for_signals.dispatch(Event::ShutdownRequested { source: "signal".into() }),
        }
    });

    let lc_for_panic = lifecycle.clone();
    std::panic::set_hook(Box::new(move |info| {
        lc_for_panic.dispatch(Event::FatalError { kind: SubsystemKind::Engine, err: info.to_string() });
    }));

    lifecycle.dispatch(Event::ProcessStarted);
    lifecycle.wait_until_terminal().await  // blocks until Failed — process::exit is called from Failed.entry
}
```
