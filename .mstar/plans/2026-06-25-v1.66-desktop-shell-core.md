## Goal

Ship `apps/desktop` ‚Äî a Tauri v2 macOS application that wraps the V1.65 `apps/web` dist, implements `TauriClient` (the desktop `NexusClient` impl, thin-over-`BrowserClient` per Phase 2b), and delivers the Q5-deferred desktop right-click actions ("Open with‚Ä¶", "Reveal in Finder") on chapter body/outline paths. The SPA code stays unchanged; only `TauriClient` swaps in. Proves the wrapper + transport end-to-end against an **externally-running** daemon (dev mode ‚Äî user runs `nexus42 daemon start`); sidecar bundling is P1.
