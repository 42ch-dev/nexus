## Goal

Converge the remaining V1.56 DF-56 (conditional routing) medium/low residuals and V1.57-new residuals that were deferred from P0's scope. This plan is the "polish pass" that closes DF-56 independent slice gaps (null semantics, converge fan-in alignment, expression routing integration tests, latency bench, eval tracing, converge timeout, loader error) and DF-56 dependent slice gaps (workspace context tracing, registry.refresh instrumentation, synthetic fallback field ambiguity, throttle-path yield, workspace_state hook activation). Also closes V1.57-new residuals: host-call smoke test un-ignore and remaining per-ID failure-path test vectors not covered by P0.

- **T7**: Resolve `R-V156P2-L004` — 0-predecessor converge loader error. Fix the error message or handling when a converge node with 0 predecessors is loaded.
- **T8**: Resolve `R-V156P3-W001` — `inject_workspace_context` zero tracing. Add `tracing::debug!` to the workspace context injection path so operators can trace what context was injected into which schedule.

- **T10**: Resolve `R-V156P3-S001` — synthetic fallback source field ambiguous. Clarify the `source` field semantics on synthetic fallback entries; add doc comment or rename the field.
- **T11**: Resolve `R-V156P3-S002` — throttle-path `.await`. Ensure the throttle path in the orchestration scheduler properly yields with `.await` (not a busy-wait or blocking call).
- **T12**: Resolve `R-V156P3-S004` — `with_workspace_state` hook activation. Activate the `with_workspace_state` hook for registry.refresh dependent edges so the refresh capability receives the current workspace context.
- **T13**: Resolve `R-V157P1-W001` — host-call smoke `#[ignore]` fix. Un-ignore the host-call smoke tests or add a documented justification comment explaining why they must remain `#[ignore]` (e.g., require live daemon + active creator).
- **T14**: Resolve remaining per-ID failure-path test vectors (`R-V157P0-L002` carry-forward). Add any missing failure-path test vectors for capability handlers not covered by P0 (focus on DF-56 dependent path handlers).
- **T15**: Resolve `R-V156P2-CACHE-01` — engine test fidelity. Audit existing convergence / conditional-routing tests to ensure they exercise real runtime paths (e.g. `record_converge_arrival`) instead of bypassing via direct unit calls. Replace any bypass paths with real-runtime E2E tests; add a test-contract comment explaining why real-runtime is required for the convergence surface.

Mitigation: assess root cause before un-ignoring; if daemon startup is the issue, fix the race or keep `#[ignore]` with a documented justification and a tracking residual.
