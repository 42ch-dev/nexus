> "The workspace `.rustfmt.toml` ignores `crates/nexus-contracts/src/generated/`"

This is **not a blocking issue** for the context-assembly feature since:
1. The diffs are exclusively in generated code
2. The AGENTS.md explicitly states these should be ignored
3. CI `verify-codegen` job validates generated file consistency
