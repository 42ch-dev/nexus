## 1. Problem Statement

Current `novel-writing` preset produces demo-grade `outline.md` + `ch01-introduction.md` + `ch02-body.md` under legacy paths. It does not implement per-chapter outline, draftÔćĺfinal status transitions, or `work-status.md` chapter table updates aligned with reference-system quality bars (distilled, not copied).

---

## 2. Goals

1. Rewrite `novel-writing` prompts to use `Works/{{work_ref}}/Outlines/chapters/ch<nn>-outline.md` and `Works/{{work_ref}}/Stories/ch<nn>-<slug>.md`.
2. Pipeline for **one chapter** (ch01 MVP): outline state Ôćĺ draft state Ôćĺ finalize state.
3. Update `work-status.md` chapter row on each transition (`outlined` Ôćĺ `draft` Ôćĺ `finalized`).
4. Integrate with FL-E `produce` stage (`creator run stage advance --stage produce` or chain default).
5. Optional light `reflection-loop` hook documented; not required for V1.36 MVP acceptance.

---

## 3. Non-goals
