## 1. Problem Statement

`Outlines/foreshadowing.md` and `event-index.md` are scaffolded with F###/E### table headers but no runtime allocates ids or promotes inline outline declarations into the index files.

---

## 2. Goals

1. Parse inline F### declarations from chapter outline "Foreshadowing Touched" section.
2. Allocate next F### id when missing; append row to `foreshadowing.md`.
3. Read index summary for prompt injection (outline/draft) when file non-empty.
4. Hermetic tests: scaffold → outline with new F001 → index file updated.
